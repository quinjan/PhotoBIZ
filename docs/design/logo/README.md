# PhotoBIZ Logo Assets

This folder contains simple SVG logo versions for the PhotoBIZ brand.

The camera and flash mark uses the same proportions across the logo set. Resize the mark as a whole when creating new variants instead of redrawing the flash, camera body, or lens independently.

- `photobiz-business-docs-logo.svg`: horizontal logo for proposals, invoices, internal docs, and slide decks.
- `photobiz-app-icon.svg`: square app icon for launchers and installable app surfaces.
- `photobiz-browser-icon.svg`: compact camera and flash favicon source.
- `photobiz-in-app-icon.svg`: compact logo for the admin shell, kiosk preview header, and in-app brand areas. It uses the same mark style as the business document logo.

Suggested usage:

- Keep `photobiz-business-docs-logo.svg` on light backgrounds.
- Use `photobiz-app-icon.svg` for 512px, 256px, 192px, and 180px exports.
- Export `photobiz-browser-icon.svg` to 32px and 16px favicon formats, but use the SVG directly where supported.
- Use `photobiz-in-app-icon.svg` inside the application so the UI does not need the heavier square app icon.
